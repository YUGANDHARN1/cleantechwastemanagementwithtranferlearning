Waste Management Classification System

Project Overview

An AI-powered waste classification system that uses deep learning to categorize waste materials into four main categories: Recyclable, Organic, Hazardous, and Non-Recyclable. The system employs VGG16 transfer learning for accurate image classification.

Technologies Used

Python 3.11 TensorFlow/Keras Flask PIL/Pillow NumPy Bootstrap 5 HTML5/CSS3 JavaScript

Key Features

  • Real-time image classification
  • VGG16 transfer learning model
  • Confidence score display
  • Multiple prediction results
  • Responsive web interface
  • Image preprocessing pipeline

Waste Categories

Recyclable
Paper, plastic, glass, metal
Organic
Food scraps, garden waste
Hazardous
Batteries, chemicals
Non-Recyclable
Composite materials
